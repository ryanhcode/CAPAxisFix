Patches various accessibility issues in cadet online testing and cadet interactive.

Fixes:

    Back to all courses button
    Tab accessibility
    Button accessibility
    Honor credit/course card accessibility
    Odometer accessibility
    Inconsistency in buttons for accessibility
    Ghost links in tests and course material
    Make score page accessible
    Make honor statement and more buttons completable

This extension has been developed & tested on the latest version of Google Chrome.

Installation Instructions:

    Extract addon ZIP
    Open google chrome and go to chrome://extensions
    Enable developer mode
    Click "Load Unpacked"
    Select the extracted folder
